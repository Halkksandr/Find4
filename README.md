## Автор
Александр Хасай

## Описание
Expense Tracker — это приложение, которое позволяет пользователям отслеживать свои личные расходы. Пользователи могут добавлять расходы, фильтровать их по категориям, а также подсчитывать общую сумму расходов за период. Данные хранятся в формате JSON.

## Запуск приложения
1. Убедитесь, что у вас установлен Python 3.
2. Установите необходимые библиотеки, если они не установлены:
   ```bash
   pip install tkinter
   Скачайте или склонируйте репозиторий:

git clone <URL вашего репозитория>
Перейдите в директорию проекта:

cd <название директории>
Запустите приложение:

python app.py
Использование
Введите сумму, категорию и дату расхода.
Нажмите кнопку "Добавить расход", чтобы сохранить запись.
Чтобы подсчитать общую сумму расходов, нажмите "Подсчитать сумму".
Используйте поле фильтра для поиска расходов по категории.
Примеры использования
Для записи еды используйте категорию "еда" с положительной суммой и корректной датой (YYYY-MM-DD).
Нажмите "Подсчитать сумму", чтобы увидеть общую сумму всех добавленных расходов.
